# Image Compressor

I built this simple Image Compressor Web App for my personal use. It allows you to upload an image, compresses it on the server using Node.js + Sharp, and lets you download the optimized version instantly. The frontend is built with HTML, CSS, and JavaScript.

## Live Web App

https://image-compressor-tawny.vercel.app/

## Features

*   Upload PNG, JPG, or JPEG images
*   Automatic server-side compression using Sharp
*   Download compressed images
*   Fast and lightweight
*   Simple and minimal UI
*   Built for personal use

## Tech Stack

### Frontend

*   HTML
*   CSS
*   JavaScript

### Backend

*   Node.js
*   Express
*   Sharp (image processing)
